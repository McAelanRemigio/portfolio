# Portfolio Website

Live site: https://McAelanRemigio.github.io/portfolio/

## Overview

This is my personal portfolio website built using a React + Vite template.

I customized the project to showcase my own work, styling, and content.

## Changes I made

* Updated content and personal information
* Modified UI and styling
* Added media assets (videos, images)
* Adjusted routing and deployment for GitHub Pages
* Fixed build and asset issues for production

## Tech Stack

* React
* Vite
* React Router
* GitHub Pages (deployment)

## Original Template

Repository: [[LINK TO ORIGINAL REPO]](https://github.com/blairxu13/persona3-website)

